Stack – Definition

A stack is a linear data structure that follows LIFO (Last In First Out) or FILO (First In Last Out) order. Insertion and deletion are allowed only at one end, called the top of the stack. :contentReference[oaicite:2]{index=2}

Basic Stack Operations

  • push(x): Insert element x at the top of the stack.
  • pop(): Remove and return the top element.
  • peek() / top(): Return the top element without removing it.
  • isEmpty(): Check whether the stack has no elements.
  • isFull(): Check whether the stack is full (for array implementation).

C Implementation – Stack Using Array

#define MAX 5
int stack[MAX];
int top = -1;

void push(int data) {
    if (top == MAX - 1) {
        printf("Stack Overflow\n");
    } else {
        top++;
        stack[top] = data;
    }
}

int pop() {
    if (top == -1) {
        printf("Stack Underflow\n");
        return -1;
    } else {
        return stack[top--];
    }
}

In the full program, you can also add peek(), display(), and a menu-driven main() as shown in your PDF notes.

Applications of Stack

  • Expression evaluation and conversion (infix, postfix, prefix)
  • Function call management (call stack)
  • Backtracking problems (e.g., maze, DFS)
  • Undo / Redo operations in editors
  • Parenthesis checking in expressions
  • String reversal

Expression Forms

  • Infix: Operator between operands – (A + B) * C
  • Prefix: Operator before operands – * + A B C
  • Postfix: Operator after operands – A B + C *

Algorithm – Infix to Postfix (Using Stack)

  1. Scan the infix expression from left to right.
  2. If the symbol is an operand, add it to the postfix string.
  3. If the symbol is (, push it onto the stack.
  4. If the symbol is ), pop from the stack and append to postfix until ( is found. Discard both parentheses.
  5. If the symbol is an operator:
    • While the top of stack has higher or equal precedence, pop and add to postfix.
    • Push the incoming operator onto the stack.
  6. After the full scan, pop all remaining operators from the stack to the postfix string.

Example – Infix: (A + B) * C

Final postfix expression: AB+C*

Postfix Evaluation – Idea

  1. Scan the postfix expression from left to right.
  2. If the symbol is an operand, push it onto the stack.
  3. If the symbol is an operator:
    • Pop the top two operands from the stack.
    • Apply the operator: result = op2 operator op1.
    • Push the result back onto the stack.
  4. At the end, the stack contains the final result.

Example: Postfix 4 5 6 * + evaluates to 34.
